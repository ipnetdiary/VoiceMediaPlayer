Available Songs

Command Command Type AudioFile

Available Commands:

  • Play - Plays the current song or starts the first song.
  • Stop - Stops the the current song and nullifies the playback time.
  • Pause - Pauses the current song.
  • Next Plays the next song in the list.
  • Previous - Plays the previous song in the list.
  • Play [slice from song's or singer's name] - Plays a song with a particular name, if found.
